        Negocio

        Google Maps

        Se usa para geocodificar direcciones y optimizar el orden de las paradas. Consigue tu API key en Google Cloud Console (activa las APIs "Geocoding API", "Directions API" y "Maps JavaScript API").

        Conexión con QuickBooks Online

        Adaptable a cualquier compañía de entregas: solo necesitas los datos de tu app de Intuit Developer. Llena el formulario y presiona "Conectar" — no hace falta tocar archivos de configuración.

        Reglas de zona Norte / Sur

        Esta app es genérica para cualquier ciudad: define la calle o dirección que separa el Norte del Sur en tu zona de reparto. Si las direcciones ya incluyen el cuadrante (NW/NE/SW/SE, típico de Miami), se detecta automáticamente sin necesidad de geocodificar.
